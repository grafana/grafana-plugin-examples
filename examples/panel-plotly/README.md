# Plotly panel

This example uses the [Plotly](https://plotly.com/) library to plot graphs.

## Dependencies

To add Plotly to a plugin, install the following dependencies.

```
yarn add react-plotly.js plotly.js
yarn add --dev @types/react-plotly.js
```
